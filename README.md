# Jarvis

Jarvis is a local-network home AI assistant monorepo.

## Project Layout

- `backend/`: FastAPI API, auth, admin user management, email, and persistence.
- `frontend/`: Nuxt 3 web app.
- `docs/`: setup, security, backend, and deployment notes.
- `database.sql`: reference PostgreSQL schema.
- `backend/migrations/`: database migration files.

This cleaned baseline removes the copied commerce and image-workflow product surfaces.
